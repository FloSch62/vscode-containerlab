About TopoViewer

Getting Started

TopoViewer provides two modes for working with Containerlab topologies:

Viewer Mode

Explore and interact with deployed topologies. Visualize running labs and manage containers.

  • Left-click: Select and view node/link details
  • Right-click node: SSH, attach shell, view logs
  • Right-click link: Capture packets with Wireshark
  • Drag: Move nodes to organize layout

Editor Mode

Create and modify topology files visually. Design your infrastructure with GUI.

  • Shift + Click canvas: Add new node
  • Shift + Click node: Start link creation
  • Alt + Click: Delete node or link
  • Right-click: Context menu for edit/delete
  • Drag node to group: Auto-assign to group

Tips

  • Use layout algorithms to auto-arrange topologies
  • Box selection to select multiple elements
  • Double-click text annotations to edit
  • Export topology as image from context menu
